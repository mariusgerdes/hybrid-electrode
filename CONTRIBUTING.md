# Contributing

Thank you for your interest in this project!

## Reporting Issues

If you find any issues with:
- The 3D models (printing problems, design flaws)
- Data inconsistencies
- Documentation errors

Please open an issue on GitHub with:
1. A clear description of the problem
2. Steps to reproduce (if applicable)
3. Your environment (printer model, software version, etc.)

## Questions

For questions about:
- **Experimental methods**: Contact marius.gerdes@kit.edu
- **3D printing**: Open an issue with the "3D models" label
- **Data interpretation**: Contact marius.gerdes@kit.edu
- **Collaboration**: Contact marius.gerdes@kit.edu

## Modifications and Improvements

We welcome improvements to:
- 3D model designs
- Documentation clarity
- Additional analysis scripts

If you've made modifications you'd like to share:
1. Fork the repository
2. Make your changes
3. Submit a pull request with a clear description

## Code of Conduct

Please be respectful and constructive in all interactions. This is an academic project aimed at advancing EEG electrode technology.
