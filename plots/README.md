# Plots Directory

This directory contains high-resolution figures from the paper.

## Available Plots

### Electrode Design
- `electrodes_combined_withtube_andlabels.png` - Overview of electrode designs (wet, dry, RPH-Trode)
- `filling.png` - Electrode filling procedure illustration
- `Electrode_Comparison.png` - Visual comparison of different electrode types

### Impedance Analysis
- `impedance_per_channel.pdf` - Individual channel impedance tracking for all 8 channels over one hour

### EEG Signal Quality
- `psd_grid_comparison.pdf/.png` - Power spectral density comparison across electrode types

### SSVEP/CCA Analysis
- `ssvep_cca_comparison.pdf` - Overall CCA correlation comparison across electrode types
- `cca_example_P004_S001_hybrid.pdf` - Example CCA correlation for RPH-Trode during motion tasks
- `cca_example_P004_S001_dry.pdf` - Example CCA correlation for dry electrode during motion tasks
